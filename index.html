Instalação

Escolha o método de instalação apropriado para sua distribuição Linux.

Arch Linux (AUR)

yay -S wine-manager

Ou compile manualmente:

git clone https://aur.archlinux.org/wine-manager.git
cd wine-manager
makepkg -si

Debian/Ubuntu

wget -c https://github.com/pedrodev2025/wine-manager/releases/download/1.2/wine-manager_1.2.deb
sudo apt install ./wine-manager_1.2.deb

Outros sistemas (Instalação manual)

sudo wget -O /usr/local/bin/wine-manager   https://raw.githubusercontent.com/pedrodev2025/wine-manager/refs/heads/main/wine-manager
sudo chmod +x /usr/local/bin/wine-manager

É recomendado instalar scripts manuais em /usr/local/bin para que não interfiram nos pacotes do sistema.

Sobre o projeto

Wine Manager é um utilitário em Bash que facilita o gerenciamento de prefixos do Wine. Ideal para entusiastas de jogos e aplicações Windows no Linux, proporcionando uma experiência mais organizada e modular.
